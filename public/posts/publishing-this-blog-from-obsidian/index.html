Publishing this blog from Obsidian

November 9, 2024 One-minute read
Patrik Björklund
Productivity

This blog is today fully published from Obsidian to GitHub Pages via the Enveloppe plugin inside Obsidian.

The plugin pushes markdown content into the Posts directory in the GitHub respository via an auto merged pull request. That then triggers a simple GitHub action that builds the static site with Hugo and deploys the output to GitHub Pages.

Super simple.
